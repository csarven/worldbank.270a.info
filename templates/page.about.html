What is this?

This site contains data from The World Bank. Currently it has World Development Indicators, World Bank Finances, and World Bank Projects and Operations data.

Technical mumbo jumbo

Process

I’ve collected the World Bank’s data in XML from their APIs, and applied XSLT 2.0 transformations to get RDF/XML. I’ve heavily made use of the standalone rapper RDF parser utility program, and your common Linux command-line scripting and tools along the way.

Apache Jena’s TDB storage system and Fuseki is used to run the SPARQL server. The HTML pages are generated by the Linked Data Pages framework, where Moriarty, Paget, and ARC2 does the heavy lifting for it.

Query

The data can be queried using the SPARQL Endpoint.

VoID

There is a VoID file. If you are here for the datadumps, please use that file to discover their locations. Statistics for the VoID file is generated using LODStats.

About the datasets

Completeness
The World Bank group is working hard to have their data available through their APIs. Sometimes things change. I'm playing safe with the Projects and Operations dataset, because its API is in testing i.e., while all the projects are listed, I'm holding back on some unstable parts of the data. Therefore, the data dumps and what’s on this site is what I consider to be safe enough. I will update accordingly.
World Development IndicatorsComplete
World Bank FinancesComplete
World Bank Projects and OperationsIncomplete

Data is only altered by removing white-space at the start and end of text content. Some of the dates in the data are transformed into IS08601 format. I will expand more on this stuff at a later date. Generally, what you probably need to know is that, only some of the data is normalized.
